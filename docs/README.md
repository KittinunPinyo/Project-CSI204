## โครงงานนี้คืออะไร

**Kick Zone (คิ๊กโซน)** ระบบเว็บไซต์ร้านค้าออนไลน์สำหรับจัดจำหน่ายรองเท้า

## 💻 ภาพรวมโปรเจค

**Kick Zone (คิ๊กโซน)** เป็นระบบ E-Commerce ที่ออกแบบมาสำหรับจัดจำหน่ายรองเท้าออนไลน์ โดยให้ลูกค้าสามารถ:

- ค้นหาและดูรายละเอียดของรองเท้าแบรนด์ต่างๆ
- หยิบสินค้าใส่ตะกร้าและดำเนินการสั่งซื้อ
- ติดตามสถานะคำสั่งซื้อของตนเอง

ทั้งนี้ **Admin (ผู้ดูแลระบบ)** สามารถเข้ามาจัดการเพิ่ม ลด แก้ไข ข้อมูลสินค้าในระบบ อัปเดตสถานะการจัดส่ง และดูภาพรวมยอดขายผ่าน Dashboard ได้แบบ Real-time

## ✨ ฟีเจอร์หลัก

### 👤 สำหรับลูกค้า (Customer)
- ✅ ค้นหาและเลือกดูสินค้ารองเท้าตามแบรนด์ พร้อมฟิลเตอร์กรองช่วงราคา
- ✅ สมัครสมาชิก และเข้าสู่ระบบ (รองรับระบบการเข้าสู่ระบบด้วยบัญชี **Google**)
- ✅ ดูรายละเอียดสินค้า ราคา โทนสี วันที่วางจำหน่าย และตรวจสอบสต็อกแบบแยกตามไซส์ (EU Size)
- ✅ เพิ่มสินค้าลงตะกร้า (Shopping Cart) และระบบบันทึกสินค้าที่ชอบ (Wishlist ❤️)
- ✅ ยืนยันการสั่งซื้อสินค้า (Checkout) พร้อมระบบ **อัปโหลดสลิปโอนเงิน**
- ✅ รองรับการใช้งาน **โค้ดส่วนลด (Promo Code)** และระบบแบนเนอร์ Flash Sale
- ✅ ระบบ **รีวิวสินค้า** (สามารถให้คะแนนดาว และแนบรูปภาพรีวิวสินค้าได้)
- ✅ ดูประวัติการสั่งซื้อ และติดตามสถานะพัสดุของตนเอง

### 🔧 สำหรับผู้ดูแลระบบ (Admin)
- ✅ **Dashboard** สรุปยอดขายรวม จำนวนออเดอร์ และจำนวนสินค้าในระบบ
- ✅ เพิ่ม / ลบ / แก้ไข ข้อมูลสินค้า (ชื่อ, แบรนด์, ราคา, รูปภาพลิงก์, โทนสี, **จัดการสต็อกแยกตามไซส์**)
- ✅ ดูรายการคำสั่งซื้อทั้งหมดจากลูกค้า และตรวจสอบความถูกต้องของสลิปโอนเงิน
- ✅ เปลี่ยนสถานะคำสั่งซื้อ *(ได้รับออเดอร์ -> กำลังจัดเตรียม -> ส่งพัสดุแล้ว -> สำเร็จเรียบร้อย)* พร้อมระบบสร้างเลข Tracking (TH-XXX) อัตโนมัติ
- ✅ **จัดการโปรโมชั่น** (สร้างโค้ดส่วนลด, กำหนดวันหมดอายุ, กำหนดจำนวนครั้งที่ใช้ได้, และเปิดโหมด Flash Sale)
- ✅ **จัดการรีวิว** (ตรวจสอบ อ่านความเห็น และลบรีวิวที่ไม่เหมาะสมออกจากระบบ)

## 🛠️ เทคโนโลยี (Tech Stack)

**Frontend (ส่วนหน้าบ้าน):**
- **React.js (Vite):** สำหรับพัฒนาส่วนติดต่อผู้ใช้งาน (UI) ให้ทำงานได้รวดเร็ว
- **React Router DOM:** สำหรับจัดการการเปลี่ยนหน้าเว็บ (Routing)
- **Bootstrap 5:** สำหรับตกแต่งความสวยงามและทำให้เว็บรองรับทุกขนาดหน้าจอ (Responsive Design)
- **Axios:** สำหรับจัดการ HTTP Requests เชื่อมต่อกับ API
- **@react-oauth/google:** สำหรับระบบยืนยันตัวตนด้วย Google

**Backend (ส่วนหลังบ้าน):**
- **Node.js & Express.js:** สำหรับสร้าง RESTful API และจัดการการทำงานของเซิร์ฟเวอร์
- **pg (node-postgres):** สำหรับเชื่อมต่อและจัดการคำสั่งฐานข้อมูล
- **Multer:** สำหรับจัดการการอัปโหลดไฟล์ (รูปภาพสลิปโอนเงิน และ รูปภาพรีวิวสินค้า)

**Database (ฐานข้อมูล):**
- **PostgreSQL (Neon):** ระบบฐานข้อมูลเชิงสัมพันธ์แบบ Cloud (Serverless Cloud Database)
